### Core - Blocks

#### Define a group of statements (i.e. code) to be executed together

##### Python (version >=0.x.x)

```py
if condition:
    # code...
```

##### Mojo (version >=0.x.x)

```mojo
if condition:
    # code...
```
...
##### Typescript (version >=0.x.x)

```ts
if (condition) {
    / code...
}
```

##### Go (version >=0.x.x)

```go
if condition {
    // code...
}
```
