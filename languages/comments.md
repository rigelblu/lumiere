### Comments

#### Write comments that will be ignored

##### Python (version >=x.x.x)

```py
# Single line comment
name = "lang" # Comment at end of line

"""
Multi-line comment
"""

'''
Multi-line comment
'''
```

##### Mojo (version >=0.x.x)

```mojo
# Single line comment
var name = "lang" # Comment at end of line

"""
Multi-line comment
"""

'''
Multi-line comment
'''
```

##### TypeScript (version >=0.x.x)

```ts
// Single line comment
name = "lang" // Comment at end of line

/* multi-line comment */

/*
 * multi-line comment
 */
```

##### Go (version >=0.x.x)

```go
// Single line comment
name := "lang" // Comment at end of line

/*
 * multi-line comment
 */
```
